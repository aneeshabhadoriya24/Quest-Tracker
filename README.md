
# 🎯 Quest Tracker

A simple and interactive web-based Quest Tracker built using HTML, CSS, and JavaScript.  
It helps users manage daily tasks, set goals, and track their progress in a clean and minimal interface

## Features

- Add daily quests/tasks
- Mark tasks as completed
- Delete completed or unwanted tasks
- Simple and responsive UI
- Built with vanilla JavaScript (no frameworks)

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
